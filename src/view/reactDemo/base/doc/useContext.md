# 型构
## 1.名称
`useContext`

## 2.参数
`MyContext`【必须是context对象本身】

什么是`context`对象？

**`context对象`是`React.createContext`的返回值**

## 3.返回值
传入的`context`对象 的当前值

# 使用时机
需要进行跨层级数据传递时。
